2 Template

Define your template by adding sections and configuring the section settings

Each section can be:

- Note (like this) section, free text

- Heading, usefull for grouping sections

- Contents like a table of contents or figures

- Data Table, with data from test results

- Data Graph, for graphical representation of test results

Each section can also have sub sections

The cover page and overall report settings can be found on the settings pane of the Report item above
